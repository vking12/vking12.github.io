Project 1 Data Explained
Data Used
The data was provided by Prof. Andrew Johnson who appears to have obtained it from Robin Wilson (January 2011). Contact information is: robin@rtwilson.com || www.rtwilson.com/academic.

Datafiles used:
choleraDeathLocations2.csv
choleraDeaths.tsv
choleraPumpLocations2.csv
naplesCholeraAgeSexData2.tsv
UKcensus1851.csv
UKcensus1851_totals.csv

Any data file that includes the number "2" on the end of it (and "_totals") have been modified by myself slightly to make them a bit easier to pipe into RStudio. In some of the csv files the latitude and longitude columns were not really defined so I defined them in “*Locations2.csv”.
