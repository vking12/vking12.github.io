project1
This project takes in data from the Cholera Outbreak in 1848 and creates a scalable user interface to explore the data in more depth. The tool itself is available on by clicking on the "explore" tab below. Description on approach, manipulation of data, source code (with readme), and interesting inferenes made through exploration of dataset (the 1840s John Snow was right).

The application has tabs on the left hand side used to navigate the different datasets from location of cholera attacks/deaths to tables to view the data in a bit more quantative way.
